Challenges and Tools Used in Our Project
Initial Challenges:
  • URL Manipulation Limitations: Initially, we tried navigating through pages by modifying the URL, but the site restricts access beyond page 5.
    (example url: https://www.rottentomatoes.com/browse/movies_at_home/genres:action?page=5)
  • Selenium Issues: Selenium was used to interact with the site and load additional pages by clicking buttons, but Google Colab restricted this approach over time.
Final Approach:
  • Pyppeteer: Successfully managed button-based page loading without issues.
  • Beautiful Soup and Requests: Utilized for HTML parsing and data extraction.
  • Memory Management: Due to RAM limitations in Google Colab, we saved the large index in chunks using Excel.

Runtime and improvements
Performance Insights:
  • Execution Time: Our queries took approximately 3 hours to execute, and indexing also took about 3 hours.
  • Time Dependency: The duration depends on the number of pages processed and the number of movies per page.
  • Load More Button: Each "Load More" button click introduces a delay due to the page's script.
  • Review Loading: Loading additional reviews also relies on the page script, adding to the delay.
  • Optimization Suggestion:
    • Start the crawler directly from page 5 (e.g., https://www.rottentomatoes.com/browse/movies_at_home/genres:action?page=5).
    • This approach would avoid the need for Pyppeteer until page 5, reducing the time spent on button clicks and utilizing only the HTML of the page.

Hubs and Authorities

In the returned pages there are 'Hubs'.
This can be seen in the way that on every page of a movie, there are the recommended movies to watch, so that every movie has access to the most recommended movies
which means that you can access these (recommended) pages from every page of a movie on rotten tomatoes.

You can see that there are also 'Authorities' in the returned pages and that are the recommended movie pages.
It seems that every recommended movie from the list can be accessed by other movie pages because the recommended movies appear on every movie page.
